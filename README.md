# Tele

A teleprompter that lives in your MacBook's notch.

![Screenshot](screenshot.png)

## Install

```
npm install
```

## Development

```
npm run dev
```

## Build

```
npm run build
```

## Run

```
npm start
```

## Shortcuts

| Action     | Default                |
|------------|------------------------|
| Play/Pause | `⌘ + Shift + S`       |
| Speed Up   | `⌘ + Shift + D`       |
| Speed Down | `⌘ + Shift + A`       |
| Seek Ahead | `⌘ + Shift + ↓`       |
| Seek Back  | `⌘ + Shift + ↑`       |

All shortcuts are configurable in Settings.
